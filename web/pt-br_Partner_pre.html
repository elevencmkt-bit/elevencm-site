Perguntas frequentes

Os computadores com Windows 10 ainda funcionarão, mas a Microsoft não fornecerá mais suporte técnico ou atualizações de segurança e recursos do Windows, o que pode deixar você exposto a um maior risco de vírus e malware.

Consulte https://www.microsoft.com/windows/windows-11-specifications para verificar os requisitos mínimos de hardware e as especificações completas para a atualização.

Uma correção é uma alteração dentro de uma versão específica de um produto Windows. Uma atualização é quando você altera as versões do sistema operacional, por exemplo, migrando do Windows 10 para o Windows 11.

É necessário verificar se seu computador atende aos requisitos mínimos abrindo Configurações > Privacidade e Segurança > Windows Update. Se o seu computador puder fazer a atualização do Windows 10 para o Windows 11 Pro, você verá uma opção no Windows Update para atualizar gratuitamente para o Windows 11 Pro.

Se o seu computador não puder atualizar para o Windows 11 Pro ou você quiser substituí‑lo, você poderá migrar para o Windows 11 Pro comprando um novo computador para empresa. Fale com um de nossos consultores.

Existem algumas maneiras de obter o Windows 11:

Comprar um novo PC com o Windows 11: o Windows 11 é a versão mais atual do Windows. Se você tiver um computador mais antigo, recomendamos que você migre para o Windows 11 comprando um computador novo. O hardware e o software melhoraram muito desde que o Windows 10 foi lançado, e os computadores de hoje são mais rápidos, mais poderosos e mais seguros.

Instalar o Windows 11 no seu PC atual: verifique se o seu PC atual atende aos requisitos mínimos abrindo Configurações > Windows Update. Se o seu PC puder atualizar do Windows 10 para o Windows 11, você verá uma opção no Windows Update para atualizar para o Windows 11 gratuitamente.

O programa Atualizações de segurança estendida (ESU) do Windows 10 oferece aos clientes a opção de receber atualizações de segurança para computadores inscritos no programa. O ESU é um programa pago que fornece a indivíduos e organizações de todos os portes a opção de estender o uso de dispositivos Windows 10 além da data de fim do suporte de forma mais segura. Saiba mais.
